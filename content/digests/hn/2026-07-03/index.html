The tool is a Python utility that preprocesses video locally by extracting only the frames that actually change (via scene detection and dedup) plus a Whisper transcript, so any LLM can "watch" without burning tokens on near-identical frames. HN quickly split into camps: a bunch of commenters pointed out that this is a misleading name because Claude still can't understand motion or object permanence from a few keyframes—Gemini's built-in video handling is both cheaper and actually token-efficient at $0.24/hour. Several people also argued that deterministic CV libraries would do a better job for tasks like reading dials or gauges, and one skeptic noted you still upload the extracted frames to Anthropic, so the "stays on your machine" claim is mostly about preprocessing. The author acknowledged that the tool's real value is token savings (90%+ reduction) rather than genuine video understanding, and a few commenters shared their own similar experiments with contact sheets or frame grids, concluding that LLMs still miss subtle motion details.
